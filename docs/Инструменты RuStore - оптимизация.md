# Оптимизация и управление приложениями через инструменты RuStore

Используйте разделИнструментыдля подключения сервисов, которые помогаютускорять

## Стоимость подключения и использования

Подключение из RuStore Консоли — бесплатнодля всех перечисленных

инструментов.

Оплата (если есть) взимаетсяв самом сервисепо его тарифам.

ПРИМЕЧАНИЕ О ПЛАТНЫХ СЕРВИСАХ

VK Реклама — оплачивается рекламный бюджет по тарифам VK.

Asodesk — 14-дневный пробный период, далее платные тарифы.

VK Карты — подключение бесплатно; возможна тарификация при высоких объемах/

коммерческом использовании согласно правилам сервиса.

GitFlic — использование по тарифному плану сервиса.

GitVerse — использование по тарифному плану сервиса.

## Инструменты разработки

VK ID— безопасная авторизация и единый вход.

Каталог SDK— централизованное управление конфигурацией SDK в приложении.

VK Карты— геолокация, маршруты, подсказки по адресам.

GitFlic— облачный Git-хостинг, публикация версий в RuStore.

GitVerse— разработка с AI-ассистентом и публикация версий в RuStore.

Когда будет полезно:разработка и поддержка, релизы, инфраструктура.

## Инструменты вовлечения пользователей

Push-уведомления— отправка сообщений для удержания и возврата пользователей.

Remote Config— управляйте конфигурацией приложения и делите пользователей на

группы для проведения А/В тестов.

VK Реклама— продвижение приложения иполучение дохода от размещения

рекламы.

Когда будет полезно:запуск и рост, удержание, промо-акции, эксперименты.

## Инструменты аналитики

Tracer— выявление сбоев и анализ причин.

Asodesk— ASO-задачи, видимость в сторах, работа с отзывами(пробный период 14

дней).

Когда будет полезно:бета/запуск (стабильность), рост (продуктовая и маркетинговая

аналитика), постоянная оптимизация.

## Подключение инструмента


1. ОткройтеRuStore Консоль.



2. Перейдите во вкладкуИнструменты.



3. Выберитеприложение, к которому нужно подключить инструмент — отобразится список


доступных сервисов.


4. Откройте нужный инструмент и следуйте инструкции по подключению.
