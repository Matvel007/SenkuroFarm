# Авторизация VK ID в вашем приложении

ИнтегрируйтеVK IDSDK в ваше приложение, чтобы установить VK ID как основной или

дополнительный способ авторизации для пользователей. Это бесплатный и безопасный

инструмент, который доступен всем разработчикам, зарегистрированным в RuStore Консоль.

## Преимущества

Аудитория VK ID составляет более 100 млн пользователей.

Авторизация в одно касание по кнопкеOne Tapидругие способы входав ваше

приложение для пользователей.

Актуальный номер телефона пользователя, который регулярно валидируется.

Возможность запросить информацию о том, подтвержден ли аккаунт пользователя и

актуальные ли в нем ФИО, пол и дата рождения.

## Интеграция VK ID SDK

Чтобы начать использовать VK ID SDK, выполните следующие действия.


1. ОткройтеRuStore Консольи переключитесь на вкладкуИнструменты.



2. Нажмите на название вашего приложения.



3. ВыберитеАвторизация VK IDв меню слева.



4. НажмитеПерейти в инструментвнизу страницы и войдите в кабинет сервиса


авторизации VK ID.


5. Зарегистрируйте свое приложение в кабинете и установите VK ID SDK. Подробности о


том, как это сделать, см. вплане интеграции VK ID.
